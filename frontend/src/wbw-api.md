# WBW Custom API Syntax (Short, Easy)

- `move x y` — Move player or object by x, y
- `jump h` — Jump with height h
- `shoot dir` — Shoot in direction (1=right, -1=left, 0=up)
- `wait t` — Wait t frames
- `if cond` — If condition (e.g. `if HP 0`)
- `else` — Else block
- `loop n` — Repeat n times
- `end` — End block
- `spawn type x y` — Spawn object
- `score +n` — Add n to score
- `sound beep` — Play sound
- `bg #222` — Set background color
- `color #fff` — Set draw color
- `enemy type x y` — Spawn enemy
- `player x y` — Set player position
- `item type x y` — Spawn item
- `msg "Hello!"` — Show message
- `goto label` — Jump to label
- `set var value` — Set variable
- `add var value` — Add to variable
- `sub var value` — Subtract from variable
- `mul var value` — Multiply variable
- `div var value` — Divide variable
- `rand var min max` — Random value
- `input key` — Check input
- `on event` — Event handler
- `off event` — Remove handler
- `win` — Win game
- `lose` — Lose game
- `draw shape ...` — Draw shape
- `rect x y w h` — Draw rectangle
- `circle x y r` — Draw circle
- `line x1 y1 x2 y2` — Draw line
- `text "str" x y` — Draw text
- `load file` — Load file
- `save file` — Save file
- `play music` — Play music
- `stop music` — Stop music
- `fx type` — Play effect
- `cam x y` — Camera position
- `zoom f` — Camera zoom
- `shake t` — Camera shake
- `fade t` — Fade screen
- `pause` — Pause game
- `resume` — Resume game
- `reset` — Reset game
- `exit` — Exit game

> All commands are extremely short and easy to read. Creators can combine them for complex logic.
